# NotABot 2.0

**English**

NotABot 2.0 is a ZK-based verification service designed to protect drops, DAOs, and much more from bot abuse. The first version of NotABot was a winner at a hackathon! ([Hackathon repository link](https://github.com/ArturInspector/mainhntrepo)).

## Features

- Zero-knowledge proof-based user verification
- **KYC per-transaction concept:** flexible, privacy-preserving verification at every transaction (without revealing personal data)
- Drop protection: prevent bot claims in airdrops and token distributions
- DAO protection: ensure genuine participation in decentralized organizations and voting
- Reliable user verification for other scenarios (events, whitelists, etc.)
- Enhanced privacy and security for all users
- Easy integration for web applications

## Getting Started

1. **Clone the repository:**
   ```
   git clone <repo_url>
   ```
2. **Install dependencies:**
   ```
   cd frontend
   npm install
   ```
3. **Start the development server:**
   ```
   npm run dev
   ```

## Project Structure

- `frontend/src/app` - Main frontend application (Next.js)
- `frontend/src/app/globals.css` - Global styles
- `frontend/tsconfig.json` - TypeScript configuration

NotABot 2.0 — это платформа верификации пользователей на базе ZK для защиты airdrop-акций, DAO и других сфер от ботов. Первая версия NotABot победила на хакатоне! ([Ссылка на репозиторий хакатона](https://github.com/ArturInspector/mainhntrepo)).

## Возможности

- Верификация пользователей с использованием zero-knowledge proof
- **KYC для каждой транзакции:** концепция гибкой и приватной проверки на каждом этапе взаимодействия (без раскрытия личных данных)
- Защита дропов: предотвращает получение airdrop-ов и токенов ботами
- Защита DAO: обеспечивает участие только настоящих пользователей в организациях и голосованиях
- Проверка пользователей в других сценариях (ивенты, вайтлисты и т.д.)
- Усиленная приватность и безопасность
- Простая интеграция для веб-приложений
